# Smartskills
## Introduction
This repository was created as part of a practical session on GitHub. The purpose of this project is to demonstrate the basic features and functions of GitHub, including repository creation, file uploading, version control, and collaboration tools.

## Objectives
- Understand the purpose of GitHub
- Learn how to create a repository
- Upload and manage files
- Edit README files
- Create branches and commits
- Explore collaboration features


## Features Covered
- Repository Creation
- README.md Editing
- File Uploading
- Folder Organization
- Branch Creation
- Commit Changes
- Basic Collaboration Tools

## Tools Used
- GitHub
- Web Browser
- Internet Connection
